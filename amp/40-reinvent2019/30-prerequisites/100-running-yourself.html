Running yourself

What are we going to do?

We are going to perform the following steps:

  • Enable AWS Config
  • Install the tools
  • Create an IAM Role

In order to run the workshop in your own account you will need to enable AWS Config and create an IAM Role named TeamRole which you must then assume in order to complete the activities.
